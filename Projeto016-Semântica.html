HTML Semântico

Antigamente, antes do HTML5 era usado a <div>, com o novo HTML, passou a se usar as tags:

<header> que indica que é um cabeçalho

<nav> que indica que é uma barra de navegação

<section> que indica que é uma seção

<article> que indica que é um artigo

<aside> que indica que é um menu lateral

<footer> que indica que é o rodapé do site





<header>: Cabeçalho do site

    Onde temos:
  • Pontos importantes do site e titulo da pagina

<nav>: Onde temos a barra de navegação

<section>: onde temos nossa seção
